# Simon Game

## Project Description

This project is a web-based implementation of the classic Simon memory game. The game tests the player's memory by generating a sequence of colors that the player must repeat in the correct order. The sequence becomes longer as the player progresses to higher levels.

## Purpose of the Project

The main purpose of this project is to practice JavaScript concepts such as DOM manipulation, event handling, and game logic while developing an interactive web application.

## Technologies Used

HTML – Used to create the structure of the web page
CSS – Used for styling and layout design
JavaScript – Used to implement the game logic and interactivity

## How to Play

1. Press any key to start the game.
2. The game will highlight a random color.
3. The player must click the same color.
4. Each level adds a new color to the sequence.
5. If the player clicks the wrong color, the game will end.

## Project Structure

index.html – Main webpage of the game
style.css – Styling of the game interface
game.js – JavaScript logic of the Simon game
